# Chaldean Numerology Name Number Calculator

A simple and easy-to-use calculator that determines a person's **Chaldean Name Number** based on the ancient Chaldean numerology system.

## ✨ Features

* Calculates the Chaldean Name Number from any name
* Uses the traditional Chaldean letter-to-number mapping
* Ignores spaces and special characters
* Provides a clean and user-friendly interface
* Fast and lightweight implementation

## 🔢 How It Works

Chaldean numerology assigns numerical values to letters according to their vibrational significance rather than alphabetical order.

| Number | Letters       |
| ------ | ------------- |
| 1      | A, I, J, Q, Y |
| 2      | B, K, R       |
| 3      | C, G, L, S    |
| 4      | D, M, T       |
| 5      | E, H, N, X    |
| 6      | U, V, W       |
| 7      | O, Z          |
| 8      | F, P          |

The calculator sums the values of all letters in the input name and returns the corresponding Chaldean Name Number.

## 🚀 Usage

1. Enter a name.
2. Click **Calculate**.
3. View the resulting Chaldean Name Number.

This project is intended for educational and entertainment purposes. Chaldean numerology is a traditional belief system and is not scientifically validated.
